## Distribution Charts

### Histogram 
Helpful for visualizing the distribution of continuous data. This visual representation groups data into bins and displays the frequency of occurence within each bin making it useful for identifying the central tendency and identifying outliers.

<b> Density plot </b>
A density plot is a smoothed version of a histogram that visualizes the distribution of a continuous variable. Unlike a histogram that bins data into grouping, a density plot estimate the concentration of values across the range to create a smooth curve to visualize data shape.


### Box plots
Also known as 'Box and Whisker' plot. Display distribution of data to compare distribution across different categories/group. The plot includes summary of statistics including the quartile (Q1, Q2 (median), and Q3), minimum, and maximum.
